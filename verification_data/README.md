# Verification Data

Отдельный набор файлов для ручной проверки проекта.

## Структура

- `text_check/operations_text_check.csv` — файл операций для проверки текстового анализа, риск-оценки и AI-комментариев.
- `document_check/operations_document_check.csv` — файл операций для проверки сопоставления с документами.
- `document_check/documents/` — документы для проверки извлечения реквизитов и сверки.
- `large_check/operations_large_5000.csv` — большой набор на 5 000 строк для проверки производительности, фильтрации и массового анализа.

## Сценарий 1. Проверка текстового анализа

Используйте:

- `text_check/operations_text_check.csv`

Рекомендуемый режим:

- можно не загружать документы;
- включить `Использовать модель для текста`, если хотите проверить генерацию комментариев через LM Studio.

Что заложено в набор:

- дубликат номера документа;
- подозрительное описание;
- нетипично большая сумма;
- пропуск обязательного поля `Описание операции`;
- редкий контрагент.

## Сценарий 2. Проверка документов

Используйте:

- `document_check/operations_document_check.csv`
- все файлы из `document_check/documents/`

Рекомендуемый режим:

- включить `Использовать модель для документов`;
- при необходимости включить `Использовать модель для текста` для генерации пояснений по найденным отклонениям.

Что заложено в набор:

- один документ без отклонений;
- документ с расхождением даты;
- документ с расхождением суммы;
- изображение для vision-разбора;
- одна операция без подтверждающего документа.

## Сценарий 3. Большой набор на 5 000 строк

Используйте:

- `large_check/operations_large_5000.csv`

Рекомендуемый режим:

- можно запускать без документов, если нужна именно нагрузочная проверка таблицы;
- можно включить `Использовать модель для текста`, но для быстрого прогона лучше оставить ее выключенной;
- для проверки производительности интерфейса документы можно не загружать.

Что заложено в набор:

- 5 000 строк;
- повторяющиеся типовые хозяйственные операции;
- редкие контрагенты;
- периодические дубликаты номера документа;
- периодические пропуски описания;
- периодические крупные суммы и подозрительные описания.
